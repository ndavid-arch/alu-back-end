API
This project contains Python scripts that consume a REST API and export TODO data.

Files
0-gather_data_from_an_API.py: Display an employee's TODO progress.
1-export_to_CSV.py: Export one employee's TODO list to CSV.
2-export_to_JSON.py: Export one employee's TODO list to JSON.
3-dictionary_of_list_of_dictionaries.py: Export all employees' TODO lists to JSON.
